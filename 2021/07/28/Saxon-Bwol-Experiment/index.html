Saxon Bwol Experiment

Experiment Name: Saxon Bowl

What it is and purposes:

The question tells that a bowl with a hole in its base will sink into the water. Saxon uses the bowl for timing purposes. The experiment wants us to find the parameters that determines the time of sinking.

Materials:

  1. Plastic bowls with holes using 3D printing

  2. Transparent container with water inside

  3. Electromagnetic releaser

  4. High-speed camera

  5. Computer apps including Tracker and Python

Requirements:

  1. To analyze the sinking process of the bowl

  2. To find the critical condition for the bowl to sink

  3. To find out how Saxon bowl is used for timing

  4. To investigate how parameters influence the time

Experiment Process

  1. Establishing a theoretical model

  2. 3D printing using CAD to model

  3. Quantitative experiment and track with Tracker

  4. Comparison between theory and experiment
